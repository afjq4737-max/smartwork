# Настройка линтеров

Проект настроен с ESLint и Prettier для обеспечения качества кода.

## Установленные инструменты

- **ESLint** - статический анализ кода
- **Prettier** - форматирование кода
- **TypeScript** - проверка типов

## Доступные команды

```bash
# Проверка кода линтером
npm run lint

# Автоматическое исправление ошибок линтера
npm run lint:fix

# Форматирование кода
npm run format

# Проверка форматирования (без изменений)
npm run format:check

# Проверка типов TypeScript
npm run type-check

# Полная проверка качества кода
npm run quality
```

## Настройки VS Code

Проект включает настройки VS Code для:
- Автоматического форматирования при сохранении
- Автоматического исправления ошибок ESLint
- Рекомендуемых расширений

## Конфигурационные файлы

- `.eslintrc.json` - настройки ESLint
- `.prettierrc` - настройки Prettier
- `.prettierignore` - файлы, игнорируемые Prettier
- `.vscode/settings.json` - настройки VS Code
- `.vscode/extensions.json` - рекомендуемые расширения

## Правила кодирования

- Используйте одинарные кавычки
- Точка с запятой обязательна
- Отступы: 2 пробела
- Максимальная длина строки: 80 символов
- Trailing comma в ES5-совместимых местах
